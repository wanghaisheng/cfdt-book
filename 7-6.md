# 39.8 综合评分算法

TypeScript：

```typescript
interface CFDTScore {

market:number;

industry:number;

company:number;

capital:number;

volume:number;

}


function calculateCFDTScore(
score:CFDTScore
){

return (

score.market*0.1+

score.industry*0.2+

score.company*0.2+

score.capital*0.25+

score.volume*0.25

);

}
```
