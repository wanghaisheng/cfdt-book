# 43.3 第一层

# Market Memory

## 市场环境数据库

---

很多交易错误：

不是股票问题。

---

而是：

环境错误。

---

例如：

同样科技股。

---

2024年：

流动性宽松。

---

2026年：

资金收缩。

---

完全不同。

---

所以记录：

每一天市场状态。

---

数据结构：

```typescript
interface MarketMemory {

date:string;


// 市场阶段

phase:

"BULL"

|
"BEAR"

|
"RANGE";


// 流动性

liquidity:number;


// 风险等级

risk:number;


// 热点方向

themes:string[];

}
```
