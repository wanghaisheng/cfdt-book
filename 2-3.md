# Purpose（本章目的）

成长股最大的利润：

通常不是来自：

追高。

而是来自：

**提前发现。**

但是：

真正的大资金：

不会告诉市场：

> "我要开始买了。"

他们通常会：

* 分批建仓
* 控制成本
* 控制价格波动
* 避免引起市场注意

因此：

在真正的主升浪开始之前，会出现一个很重要的阶段：

> **资金已经进入，而市场还没有形成共识。**

CFDT：

把这个阶段定义为：

**Capital Accumulation（资金潜伏期）**

这是：

整个系统：

收益风险比：

最高的阶段。

---

# 6.1 为什么机构不会一天买完？

假设：

基金：

准备：

买：

10亿元。

如果：

一天：

全部买。

会发生：

```text
价格上涨

↓

成本提高

↓

更多资金跟风

↓

建仓失败
```

因此：

机构：

一定会：

慢慢买。

所以：

真正的大资金：

通常留下：

三个痕迹。

第一：

资金：

持续流入。

第二：

价格：

涨得不多。

第三：

成交额：

缓慢增加。

---

# 6.2 真正的潜伏期是什么样？

很多人认为：

吸筹就是：

横盘。

其实：

远远不够。

CFDT：

定义：

真正的资金潜伏：

必须同时满足：

五个条件。

---

## 第一：

产业方向正确

产业：

必须：

值得：

长期配置。

例如：

AI。

电子特气。

国产替代。

机器人。

如果：

行业：

本身：

没有：

成长空间。

资金：

很难：

持续：

进入。

产业：

永远：

排第一。

---

## 第二：

资金持续流入

不是：

一天。

而是：

持续。

例如：

最近：

7天。

10天。

20天。

资金：

保持：

净流入。

注意：

CFDT：

更关注：

持续性。

而不是：

绝对值。

连续：

10天：

每天：

3000万。

往往：

比：

一天：

3亿。

更有价值。

---

## 第三：

价格保持克制

真正的潜伏：

价格：

通常：

不会：

快速上涨。

例如：

10天：

资金：

累计：

流入：

5亿元。

价格：

上涨：

6%。

这是：

健康。

如果：

价格：

已经：

上涨：

40%。

那么：

可能：

已经：

不是：

潜伏。

而是：

拉升。

---

## 第四：

成交额缓慢增加

这里：

与你原来的：

成交额跃迁：

形成：

对应。

Stage1：

成交额：

温和增加。

Stage2：

成交额：

数量级跃迁。

不要：

混淆。

因此：

CFDT：

把：

成交额：

分成：

两个阶段。

---

### Stage1

```text
2亿

↓

3亿

↓

4亿
```

温和增加。

---

### Stage2

```text
4亿

↓

15亿
```

跃迁。

所以：

潜伏：

不是：

没有成交。

而是：

成交：

慢慢增加。

---

## 第五：

筹码越来越稳定

真正：

机构：

建仓。

通常：

会出现：

两个现象。

上涨：

放量。

下跌：

缩量。

横盘：

波动：

越来越小。

说明：

筹码：

逐渐：

集中。

---

# 6.3 为什么不能只看资金流？

这是：

很多交易者：

最容易：

犯错。

例如：

一天：

主力：

流入：

5亿元。

马上：

买。

实际上：

可能：

只是：

消息刺激。

或者：

游资。

因此：

CFDT：

强调：

资金：

必须：

结合：

位置。

例如：

如果：

股票：

已经：

上涨：

150%。

即使：

继续：

流入。

意义：

也完全不同。

因此：

资金：

永远：

不能：

脱离：

生命周期。

---

# 6.4 真吸筹 vs 假吸筹

这是我们之前重点讨论的内容。

| 特征  | 真资金潜伏      | 假流入 / 诱多 |
| --- | ---------- | -------- |
| 位置  | 底部、平台、趋势初期 | 高位连续上涨后  |
| 资金  | 连续流入       | 单日爆发     |
| 价格  | 温和上涨       | 快速拉升     |
| 成交额 | 缓慢增加       | 突然放大     |
| 持续性 | 数周         | 一两天      |

这里有一个重要原则：

> **资金持续性，比资金强度更重要。**

# 6.5 资金潜伏评分（Accumulation Score）

这一部分，把我们之前讨论正式标准化。

| 项目      | 权重 |
| ------- | -: |
| 产业景气度   | 20 |
| 连续资金流入  | 30 |
| 价格克制    | 20 |
| 成交额温和增加 | 15 |
| 筹码稳定    | 15 |

总分：

100分。

评分建议：

| 分数     | 动作    |
| ------ | ----- |
| 85~100 | 建立观察仓 |
| 70~85  | 持续跟踪  |
| <70    | 放弃    |

注意：

这里只允许：

观察仓。

不是：

核心仓。

# Core Principle

资金：

真正有价值的时候：

不是：

市场已经发现。

而是：

市场还没有发现。

CFDT：

寻找：

不是：

热门股。

而是：

**即将成为热门股。**

# Rules

建立：

资金潜伏池。

每天：

筛选：

满足：

资金潜伏评分：

85分以上。

仓位：

20%。

下一阶段：

成交额跃迁。

# Don't

不要：

* 因为一天资金流入就判断吸筹。
* 在高位把资金流入误判为潜伏。
* 因为价格不涨就否定资金进入。
* 用同一个标准判断潜伏期和主升期。

4. 当前价格是否仍然克制？
