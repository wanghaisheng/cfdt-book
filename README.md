# Introduction

CFDT Trader OS 是一套面向科技成长股交易者设计的完整交易操作系统。

它不是一个预测股票涨跌的模型，也不是一个简单的量化选股工具。

它的核心目标：

> 将优秀交易者长期积累的市场认知、产业理解、资金判断和风险纪律，转化为一个可执行、可复盘、可进化的 AI 辅助交易系统。

# CFDT核心理念

## 1. 不预测市场

CFDT不回答：

> 明天哪只股票上涨？

而回答：

> 当前市场中，哪些资金行为值得研究？

---

## 2. 不追逐价格

关注：

```

资金流

成交额变化

产业趋势

市场环境

```

---

## 3. 先控制风险，再寻找机会

交易公式：

```

长期收益

=

机会捕捉能力

×

风险控制能力

```

# 核心交易纪律

CFDT最终形成：

## Rule 001

资金行为比价格更重要。


## Rule 002

成交额跃迁代表注意力变化。


## Rule 003

单日资金流入不是机构行为。


## Rule 004

高收益必须匹配风险控制。


## Rule 005

创新高之后优先保护资本。


## Rule 006

每一次亏损必须产生新规则。
