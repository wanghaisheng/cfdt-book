# Account Risk

## 账户风险引擎

这是你的核心需求。

---

建立：

# HWM系统

High Water Mark。

---

每日记录：

```typescript
interface AccountSnapshot {

date:string;

equity:number;

hwm:number;

drawdown:number;

}
```

---

计算：

```typescript
function calculateDrawdown(
equity:number,
hwm:number
){

return (

hwm-equity

)/hwm;

}
```

---

173万。

---

现在：

96万。

---

系统输出：

```text

173万


Current:

96万


Drawdown:

44.5%


Risk Level:

CRITICAL
```
